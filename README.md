# Sudoku Game

A simple Sudoku game built with Python and Tkinter.

## Features

- Graphical user interface
- Sudoku solver
- New game button
- Check if solved

## How to Run

1. Ensure Python 3 is installed.
2. Run `python main.py`

## Requirements

- Python 3.x
- Tkinter (usually included with Python)

## Usage

- Fill in the numbers 1-9 in the grid.
- Use the Solve button to auto-solve.
- Use New Game to start over.
- Use Check to see if completed.
